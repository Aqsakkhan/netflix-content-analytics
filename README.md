# Netflix Content Analytics

## About the Project

In this project, I analyzed Netflix's content library using Python and data visualization.

The main goal was to explore Netflix data and find useful insights about movies, TV shows, genres, ratings, countries, directors, and content growth over time.

This project helped me practice data cleaning, exploratory data analysis (EDA), and creating visualizations using Python.

---

## Dataset Information

- Dataset: Netflix Movies and TV Shows
- Total Records: 8,800+ titles
- Features Used:
  - Title
  - Type
  - Director
  - Country
  - Release Year
  - Rating
  - Genre
  - Date Added

---

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Findings

- Movies make up most of Netflix's content library.
- The United States contributes the highest number of titles.
- Netflix experienced major growth between 2016 and 2019.
- TV-MA and TV-14 are the most common content ratings.
- International Movies and Dramas are among the most popular genres.
- Netflix offers content from many countries and directors, creating a diverse library.
